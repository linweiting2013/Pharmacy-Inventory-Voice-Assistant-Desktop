# 🏥 醫院藥局語音盤點助理 (Inventory Voice Assistant)

[![🏥 醫院藥局語音盤點助理 - 示範影片](https://img.youtube.com/vi/wtBUhhXFABY/maxresdefault.jpg)](https://youtu.be/wtBUhhXFABY?si=o9Pm0oG48llfc2qu)

*🎬 [點此在 YouTube 觀看測試與示範影片](https://youtu.be/wtBUhhXFABY?si=o9Pm0oG48llfc2qu)*

這是一款專為醫院藥局設計的桌面端語音應用程式。透過串接 Google 最先進的 **Gemini 3.1 Flash Live API** 與 **Google Sheets API**，能自動透過自然語言對話與藥師互動，聽取盤點數量並即時更新至雲端試算表。

它成功將傳統「兩人一組（一人報數、一人抄寫）」的盤點作業，簡化為「一人語音獨立作業」，大幅釋放藥師人力。

---

## ✨ 系統特色功能

- **🤖 語音引導流程 (System-Led)**：助理會自動讀出試算表上第一筆未盤點的藥物，等待藥師回報數量。
- **📝 智慧單位辨識**：支援複雜的醫療計量單位口語表達（例如：「三盒兩片又六顆」、「十七盒零五顆」）。
- **📊 雲端即時同步**：語音確認後，無縫將數據寫入 Google 試算表對應的欄位，免去後續手動騰打。
- **🔇 軟體防回音 (半雙工模式)**：支援直接使用電腦外放喇叭，助理講話時會自動靜音麥克風，防止語音自我打斷。
- **🗂️ 多調劑台併發作業**：支援讀取試算表中的所有分頁（工作表），不同藥師可各自選擇不同的「調劑台」分頁同時進行盤點，互不干擾。
- **💊 聰明例外處理**：
  - **找不到藥**：可直接問「這代碼是什麼？」或「這放在哪裡？」，助理會立刻查表回答。
  - **更新單位含量**：如果需要更新包裝單位，例如對助理說「把這個藥的包裝改為每盒50顆」或「更新單位含量為每片10顆」，助理會自動更新對應藥品的 G 欄（每盒幾顆）或 H 欄（每片幾顆），且會保持其餘數量與欄位不變。
  - **帳外藥品**：遇到試算表上沒有的藥，只要說「我要新增藥品」，助理會依序跟您索取代碼、名稱、數量與儲位，並自動建檔在試算表最下方。
  - **口誤更正**：如果講錯數量，直接說「不好意思我剛剛講錯了，應該是...」，助理具備強大的上下文理解能力，能正確覆寫。

## 💻 系統需求

- **作業系統**：Windows 10 / 11 (一般醫院電腦即可，免安裝系統級服務)。
- **環境**：已安裝 Python 3.10 或以上版本。
- **硬體**：可用的麥克風與喇叭 (建議使用藍牙耳機麥克風以獲得最佳收音體驗，但亦支援直接外放喇叭)。

---

## 🛠️ 步驟一：環境安裝

1. 將本專案資料夾放在您的電腦中 (例如 `D:\A\Projects\盤點語音助理`)。
2. 開啟 Windows 命令提示字元 (cmd) 或 PowerShell。
3. 切換至本資料夾目錄：
   ```bash
   cd D:\A\Projects\盤點語音助理
   ```
4. 安裝所需的 Python 套件：
   ```bash
   pip install -r requirements.txt
   ```

---

## 🔑 步驟二：Google 試算表權限設定

為了讓程式能夠自動幫您寫入資料，您需要設定「Service Account (服務帳戶)」。這是一把讓程式有權限修改特定試算表的專用鑰匙。

1. **取得 Service Account 金鑰**：
   - 至 Google Cloud Console 啟用 Google Sheets API 與 Drive API。
   - 建立一個 Service Account，並匯出/下載 JSON 格式的金鑰檔案。
   - 將該 JSON 檔案妥善保存在您的電腦中（建議存放在本專案目錄下）。
2. **分享試算表編輯權限**：
   - 打開該 JSON 檔案，找到 `client_email` 欄位（格式如 `xxx@yyy.iam.gserviceaccount.com`）。
   - 打開您用來盤點的 Google 試算表，點擊右上角的「共用」。
   - 將該 Email 貼上，並授予 **「編輯者」** 權限。

### 📋 試算表欄位格式建議
為確保工具正常運作，試算表建議包含以下依序的欄位結構：
A: `藥品代碼`、B: `藥品名稱`、C: `藥品儲位`、D: `數量(盒/罐/束)`、E: `數量(片)`、F: `數量(顆)`、G: `單位含量(每盒幾顆)`、H: `單位含量(每片幾顆)`、I: `總數量(公式 = D2*G2 + E2*H2 + F2)`。

---

## 🚀 步驟三：啟動與設定

1. 執行以下指令啟動圖形介面：
   ```bash
   python main.py
   ```
2. 首次使用時，請點擊 **「系統設定」** 頁籤，並依序填寫：
   - **Gemini API Key**：請至 Google AI Studio 申請免費的 API 金鑰並貼上。
   - **Service Account JSON**：點擊「瀏覽...」選擇步驟二取得的 JSON 金鑰檔。
   - **Google 試算表網址/ID**：貼上您要盤點的 Google 試算表完整網址。
3. **選擇工作表 (調劑台)**：
   - 輸入完金鑰與網址後，點擊 **「取得工作表清單」**。
   - 程式會自動連線並在下拉選單中列出該試算表的所有分頁（例如：調劑台A、調劑台B）。
   - 選擇您這台電腦負責盤點的分頁。
4. 點擊畫面最下方的巨大 **「儲存設定」** 按鈕。

---

## 🎙️ 步驟四：開始盤點

1. 切換至 **「盤點作業」** 頁籤。
2. 選擇您要使用的 **麥克風** 與 **喇叭/耳機** 裝置。
   - *提示：如果使用喇叭外放，請確保右方的「防喇叭回音 (半雙工)」有打勾。*
3. 點擊綠色的 **「▶ 開始連線」**。
4. 聽到「滴」聲後，助理會主動讀出第一筆尚未盤點的藥品名稱。
5. 請走到該藥品儲位，清點後用語音回報（例如：「這裡有十七盒又五片」）。
6. 助理複誦確認無誤後，畫面日誌會顯示「更新數量中...」，接著助理會馬上唸出下一筆藥品，如此不斷循環直到盤點完成！

## 📖 疑難排解 (Troubleshooting)

- **Q: 助理一直瘋狂重複講同一句話 (例如：掌聲鼓勵)？**
  - **A**: 這是舊版 Gemini Native 模型的幻覺 Bug。本系統已預設採用最穩定的 `gemini-3.1-flash-live-preview` 模型，如果遇到異常，請嘗試刪除根目錄的 `config.json` 讓系統重新載入預設的防呆提示詞。
- **Q: 用喇叭放出來時，助理講一半常常會突然自己斷掉停住？**
  - **A**: 麥克風把助理的聲音收進去，讓它以為您在插嘴。請勾選介面上的 **「防喇叭回音 (半雙工)」**，或者改戴耳機。
- **Q: 發現錯誤 `1011 None. Internal error encountered`？**
  - **A**: 這通常發生在 Google 伺服器端。請確認您報數量時是用「數字」的概念，我們已經在系統加入了嚴格的強制阿拉伯數字輸出機制來防堵此崩潰。
